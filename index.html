Área de Experticia

Reporte de Errores

Generando informes claros y procesables en JIRA, gestionando tickets para asegurar el seguimiento y la rápida resolución de incidencias críticas, mejorando la calidad y eficiencia del software.

QA Automation

Implementando soluciones de automatización de pruebas para aplicaciones web y móviles con Playwright y TypeScript, mejorando la eficiencia y cobertura de pruebas mientras optimizo el ciclo de vida del software.

API Testing

Realizando pruebas exhaustivas de APIs con Postman, Playwright y JMeter para asegurar fiabilidad, funcionalidad y el rendimiento de los servicios bajo diferentes condiciones de carga y estrés. Esto incluye la validación de endpoints, la integración con otros servicios y la seguridad de las interfaces.

Performance Testing

Evaluando el rendimiento de aplicaciones con JMeter, incluyendo pruebas de carga, estrés y usabilidad, para asegurar estabilidad, tiempos de respuesta y experiencia del usuario.
